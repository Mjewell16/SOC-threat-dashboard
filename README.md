# AI-Driven SOC Threat Detection Dashboard

A Streamlit-based cybersecurity dashboard that simulates and visualizes live security events, authentication attempts, brute force attacks, and global threat activity.

## Features
- Live event monitoring
- Threat level calculation
- Authentication portal simulation
- Interactive world attack map
- Brute force detection
- CSV log ingestion
- Real-time dashboard metrics

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- Git/GitHub

## Project Goals
This project was built to demonstrate:
- SOC analyst concepts
- Threat monitoring workflows
- Security event visualization
- Python automation
- Cybersecurity dashboard design

## Future Improvements
- SIEM integrations
- Machine learning threat scoring
- Real API log ingestion
- Cloud deployment
- User authentication improvements

## Running the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
